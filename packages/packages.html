This document defines the set of prefixes that Reading Systems are expected to recognize when used in EPUBÂ® Package Document metadata.

This document is subject to change at any time. The prefixes defined herein will never be removed, but may be deprecated.

Any use of the prefixes marked [draft] should be considered experimental.

Deprecated prefixes are identified by the label [deprecated]. These prefixes are no longer recommended for use.

Prefixes without a label are considered stable.

Prefixes

The following prefixes are defined for use in package metadata.

For more information about prefixes, including how to declare and use them in the Package Document, please refer to Vocabulary Association Mechanisms in the Publications specification.
